文件行为指标
模型与动画均通过原生文件选择器打开，只默认筛选 .xml。
UTF-8 与带 BOM 的 UTF-8 XML 均能读取；返回前移除 BOM。
另存为未填写 .xml 时自动补全扩展名。
保存内容按 UTF-8 写入，不改变前端序列化结果。
覆盖保存只允许本次会话中由用户打开或另存过的 XML 路径。
用户取消打开/保存时返回 null，不得改变当前模型、动画或脏状态。
浏览器开发模式保留文件输入与下载回退，不依赖 Tauri 全局对象。
模型兼容指标
alpha 为 0 的透明体素不得被转换为不透明。
移动体素后骨骼绑定仍依赖稳定体素身份。
删除体素时同步移除对应绑定索引。
重新绑定按最近骨骼约束工作。
读写模型后，非体素 XML 内容在语义上保持。
新建模型能生成 1 个或 8 个中心基点体素。
动画兼容指标
读取动画名称、循环、结束时间、速度及每帧骨骼点坐标。
预览时绑定体素随动画骨骼约束移动。
编辑并导出后保持原动画 XML 层级与关键属性。
只有“创建与编辑”页允许在场景中拾取和拖动骨骼点；预览页不得修改关键帧。
骨骼拖动更新当前关键帧坐标、坐标输入框和未导出状态，切换关键帧后结果仍保留。
交互回归指标
WASD 使用完整摄像机局部方向，俯仰时 W/S 包含垂直分量。
A/D 使用摄像机局部左右方向；Shift 保持双倍速度。
场景外的按钮、下拉框和非文字控件获得焦点时，WASD 仍控制视角；文字输入框不得抢占 WASD。
雕刻、绘色和取色页共享应用内色盘；点击色块开关色盘，体素取色后工具模式保持不变。
首次操作风格选择只改变左键旋转模式。
字体、主题、快捷键和覆盖确认设置可持久化。

各目标系统至少完成一次 Tauri 启动、打开模型、另存为和覆盖保存检查。
Windows 安装包在缺少开发工具的普通用户环境中安装并启动。
跨平台差异允许范围
文件对话框外观、默认目录和路径分隔符由操作系统决定。
Windows 使用 WebView2，macOS 使用 WKWebView，Linux 使用 WebKitGTK；允许字体栅格和细微布局像素差异。
不允许因平台差异改变 XML 内容、体素坐标、骨骼绑定或动画数值。
